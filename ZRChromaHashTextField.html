ZRChromaHashTextField
A sexy, secure visualization of password field input
Project page: http://github.com/itszero/ZRChromaHashTextField

Introduction

The Chroma-Hash is originally a jQuery plugin which provided a dynamic visualization of text-field value using ambient color bars. Now, it has been ported to Cappuccino framework. All you have to do is use ZRChromaHashTextField, and voila, now your password input field has such feature. :D

Features

  • A fancy, yet useful secure password visualization
  • Let your user check their input of password by the color bars
  • Support salt, make it even secure
  • We do a smooth animation
  • Make it garyscale if the password length is too short.
